# Interactive Portfolio Website

## Project Overview
This project is an interactive portfolio website created using HTML, CSS, and JavaScript. The purpose of this project was to understand the practical application of JavaScript and create a responsive and interactive webpage.

## Features
It contains a header, navigation menu, about section, skills section, projects section, contact form, dark mode toggle, animations, form validation, local storage support, hover effects, and footer.

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## JavaScript Features
- DOM Manipulation

- Form Validation
- Dark Mode Toggle
- Local Storage
- Console Debugging
- Animations and Transitions

## Learning Outcome
This project helped me understand and apply the concepts of JavaScript, DOM manipulation, event handling, form validation, and local storage while creating an interactive portfolio website.
